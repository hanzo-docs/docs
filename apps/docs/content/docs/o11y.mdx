# Hanzo O11y

Unified observability for the Hanzo platform. Prometheus metrics, Grafana dashboards, OpenTelemetry tracing, Loki log aggregation, alerting, and SLO management.

## Highlights

- **Prometheus Metrics** -- 15-second scrape interval, 30-day retention, PromQL
- **Grafana Dashboards** -- Pre-built dashboards for every layer of the stack
- **Distributed Tracing** -- OpenTelemetry-native with automatic context propagation
- **Log Aggregation** -- Structured log ingestion and LogQL queries via Loki
- **Alerting** -- Threshold, anomaly, and SLO-burn-rate alerts to PagerDuty, Slack, webhooks
- **SLO Management** -- Define, track, and alert on Service Level Objectives

**Endpoint:** `o11y.hanzo.ai`
**Prometheus:** `o11y.hanzo.ai:9090`
