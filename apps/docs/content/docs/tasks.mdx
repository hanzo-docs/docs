# Hanzo Tasks

Durable workflow execution engine for building invincible distributed applications. Define complex business logic as code with automatic retries, saga compensation, and exactly-once semantics.

## Highlights

- **Durable Execution** -- Workflows survive crashes, restarts, and deployments
- **Automatic Retries** -- Configurable retry policies with exponential backoff
- **Saga Compensation** -- Compensating actions for partially-failed multi-step workflows
- **Scheduled Workflows** -- Cron and interval scheduling with guaranteed execution
- **Multi-Tenant** -- Each Hanzo org maps to an isolated namespace
- **SDKs** -- Go, Python, TypeScript, Java

**Endpoint:** `tasks.hanzo.ai` (UI) / `tasks-api.hanzo.ai:443` (gRPC)
