# Hanzo Dev

**Build, ship, and iterate faster** with the most popular open source coding agent. Hanzo Dev brings AI-powered development to your terminal with 500+ models, cloud agents, code reviews, and more.

```bash title="Install"
npx -y @hanzo/dev
```

## Why Hanzo Dev?

- **Stay in your flow** — Sync across every interface, persist your context, ship without friction
- **1000x every dev** — Multiply your output, code at thought speed, use the best model for the job
- **Accelerate AI ROI** — Stop AI sprawl, track and improve velocity, centralize management
