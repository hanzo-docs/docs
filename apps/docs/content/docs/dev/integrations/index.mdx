Hanzo Dev integrates with editors and developer tools through the Agent Communication Protocol (ACP) and MCP.

## Available Integrations

- [Zed Editor](/docs/dev/integrations/zed) — ACP integration for the Zed editor
- [GitHub Copilot](/docs/dev/integrations/copilot) — CLI-based Copilot features

## MCP Server

Hanzo Dev can also run as an MCP server for use with any MCP-compatible client:

```bash
dev mcp
```
