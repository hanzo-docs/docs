Hanzo Dev is an AI-powered coding assistant that runs in your terminal. It provides interactive TUI sessions, non-interactive execution for CI/CD, multi-agent orchestration, and sandboxed command execution.

## Quick Start

```bash tab="npm"
npm install -g @hanzo/dev
```

```bash tab="Homebrew"
brew tap hanzoai/tap
brew install hanzo-dev
```

```bash tab="From source"
git clone https://github.com/hanzoai/dev.git
cd dev/codex-rs
cargo build --release
```

Then launch:

```bash
dev "explain this codebase to me"
```
