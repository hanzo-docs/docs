## Skills

Skills extend Hanzo Dev's capabilities with specialized behaviors and tool integrations.

### Managing Skills

- Open the skills manager with `/skills` in the TUI
- Enable or disable available skills
- Configure skill-specific settings

### Usage

Skills are loaded into the agent context and provide additional tools and instructions for specific tasks. Manage them through:

- The `/skills` slash command
- The Settings overlay (Skills section)
