## Custom Prompts

Hanzo Dev supports saving and managing custom prompt snippets for reuse.

### Managing Prompts

- Open the prompt manager with `/prompts` in the TUI
- Create, edit, and delete prompt snippets
- Prompts are stored in `~/.hanzo/config.toml`

### Usage

Access saved prompts through:
- The `/prompts` slash command
- The Settings overlay (Prompts section)

Custom prompts are injected into the conversation context when selected, making it easy to reuse common instructions across sessions.
