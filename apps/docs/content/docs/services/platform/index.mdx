# Hanzo Platform

Hanzo Platform provides a PaaS API for managing deployments, containers, databases, domains, and infrastructure.

**Endpoint:** `platform.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/platform/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/platform/projects
```

## Key Features

- Application deployments with Git integration
- Container management
- Managed databases (Postgres, Redis, MongoDB)
- Custom domain configuration
- Environment variable management
