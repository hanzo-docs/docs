Hanzo Platform includes **Visor**, a unified virtual machine manager that lets you launch, monitor, and connect to VMs across multiple cloud providers without leaving the dashboard.

## Supported Providers

| Provider | Instance Types | Regions | Key Features |
|----------|---------------|---------|--------------|
| AWS EC2 | 400+ | 30+ | Broadest selection, spot instances |
| DigitalOcean | 20+ | 14 | Simple pricing, fast provisioning |
| Hetzner Cloud | 15+ | 5 | Best price/performance ratio |

## Capabilities

- **One-Click Launch** -- Select a provider, pick a size, and deploy in under a minute
- **Browser Terminal** -- SSH into any VM directly from your browser via Apache Guacamole
- **Block Storage** -- Create, attach, resize, and snapshot volumes
- **SSH Key Management** -- Upload and assign keys across providers from a single keyring
- **Monitoring** -- CPU, memory, disk, and network metrics streamed in real time

## How It Works

<Steps>

### Connect a Cloud Account

Navigate to **Settings > Cloud Providers** and add your API credentials for AWS, DigitalOcean, or Hetzner.

### Launch a VM

Go to **Virtual Machines > New** and select your provider, region, instance type, and OS image. Click **Create**.

### Connect

Once the VM is running, click **Terminal** to open a browser-based SSH session, or copy the IP to connect with your local client.

</Steps>

<Callout type="info">
All VM operations are logged in the audit trail. Team members need the **Developer** role or above to create and manage VMs.
</Callout>
