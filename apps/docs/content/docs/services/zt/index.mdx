# Hanzo ZT

Hanzo ZT provides programmable zero-trust networking for AI infrastructure. Build secure overlay networks with identity-based access control, no inbound ports, and end-to-end encryption.

**Endpoint:** `zt.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/zt/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/zt/networks
```

## Key Features

- Zero-trust overlay networking
- Identity-based access control
- No inbound ports required
- End-to-end encryption
- Service mesh for AI workloads
