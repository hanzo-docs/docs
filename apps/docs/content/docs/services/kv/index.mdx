# Hanzo KV

Hanzo KV provides a managed key-value store with caching, pub/sub messaging, and stream processing.

**Endpoint:** `kv.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/kv/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  -X POST https://api.hanzo.ai/v1/kv/set \
  -d '{"key": "session:abc", "value": "data", "ttl": 3600}'
```

## Key Features

- Low-latency key-value storage
- TTL-based expiration
- Pub/sub messaging
- Stream processing
- Sorted sets and data structures
