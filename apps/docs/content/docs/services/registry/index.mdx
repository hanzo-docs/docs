# Hanzo Registry

Hanzo Registry provides a container image registry with vulnerability scanning, image signing, access control, and replication.

**Endpoint:** `registry.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/registry/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/registry/repositories
```

## Key Features

- OCI-compliant container registry
- Automated vulnerability scanning
- Image signing and verification
- Role-based access control
- Cross-region replication
