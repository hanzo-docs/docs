---
title: "Nuxt"
description: "How to use Hanzo KMS to inject environment variables and secrets into a Nuxt app."
---

Prerequisites:

- Set up and add envars to [Hanzo KMS Cloud](https://app.kms.hanzo.ai)
- [Install the CLI](/cli/overview)

## Initialize Hanzo KMS for your [Nuxt](https://nuxtjs.org) app

```bash
# navigate to the root of your of your project 
cd /path/to/project

# then initialize kms
kms init
```

## Start your application as usual but with Hanzo KMS

```bash
kms run -- <your application start command>

# Example
kms run -- npm run dev
```
