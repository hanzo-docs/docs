# Hanzo Visor

Hanzo Visor is a VM and remote session management platform. It manages cloud machines, remote desktop sessions via Guacamole (RDP/SSH/Telnet), and environment provisioning.

**Endpoint:** `visor.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/visor/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/visor/machines
```

## Key Features

- Cloud VM provisioning and management
- Remote desktop sessions (RDP, SSH, Telnet)
- Environment templates
- Session recording and playback
- Multi-cloud support
