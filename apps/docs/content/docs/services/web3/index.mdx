Hanzo's Web3 infrastructure is powered by **Bootnode** — a unified SaaS control plane for all blockchain operations. It provides multi-chain RPC, indexed data APIs, smart wallet abstraction, fleet management, and full observability.

## Surfaces

| Surface | URL | Purpose |
|---------|-----|---------|
| **Lux Cloud** | [cloud.lux.network](https://cloud.lux.network) | Ops dashboard with full observability |
| **Lux Build** | [lux.build](https://lux.build) | Developer portal |
| **Bootnode Docs** | [docs.bootno.de](https://docs.bootno.de) | Full API documentation |

## Capabilities

- **Multi-chain RPC** — JSON-RPC and WebSocket for 30+ blockchains
- **Token & NFT APIs** — Indexed ERC-20 balances, NFT metadata, transfers
- **Smart Wallets** — ERC-4337 account abstraction with gas sponsorship
- **Webhooks** — Real-time event streaming for on-chain activity
- **Fleet Management** — Deploy and scale validator nodes per-network
- **Infrastructure Services** — Block explorers, bridges, DEXs, multisig wallets, faucets
- **Observability** — Health, metrics, alerts, logs via ClickHouse + Loki + Grafana
- **ZAP Protocol** — Cap'n Proto binary RPC for AI agent infrastructure control
