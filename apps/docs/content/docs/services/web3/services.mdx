Bootnode manages 22 infrastructure service types covering the full blockchain stack.

## Service Categories

| Category | Services |
|----------|----------|
| **Core** | api, web, indexer, bundler, webhook-worker |
| **Blockchain** | explorer, graph-node, bridge, exchange, validator, rpc-proxy, staking, market, faucet |
| **Security** | safe, safe-transaction, safe-config, mpc, mpc-postgres, gateway |

## Operations

- **Deploy**: `POST /v1/infra/services/{service}/deploy`
- **Scale**: `POST /v1/infra/services/{service}/scale`
- **Logs**: `GET /v1/infra/services/{service}/logs`
- **Status**: `GET /v1/infra/services`

Each service can be deployed per-network (mainnet/testnet/devnet) per-organization.

See [docs.bootno.de/docs/services](https://docs.bootno.de/docs/services) for full reference.
