# Hanzo Console

Hanzo Console is an LLM engineering platform for observability, prompt management, evaluations, and datasets. It provides comprehensive tracing and monitoring for AI applications.

**Endpoint:** `console.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/console/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/console/traces
```

## Key Features

- LLM observability and tracing
- Prompt management and versioning
- Evaluation pipelines
- Dataset management
- Cost tracking and analytics
