# Hanzo DB

Hanzo DB provides serverless PostgreSQL with instant branching, autoscaling compute, and point-in-time restore.

**Endpoint:** `db.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/db/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/db/projects
```

## Key Features

- Serverless PostgreSQL with autoscaling
- Instant database branching for dev/test
- Point-in-time restore
- Connection pooling
- Read replicas
