# Hanzo Search

Hanzo Search is a high-performance, typo-tolerant search engine built in Rust. It provides instant search, faceted filtering, and AI-powered ranking for any dataset.

**Endpoint:** `search.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/search/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/search/indexes/products/search?q=laptop
```

## Key Features

- Instant search with sub-50ms response times
- Typo-tolerant and fuzzy matching
- Faceted filtering and sorting
- AI-powered relevance ranking
- Multi-index support
