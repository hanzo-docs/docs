# Hanzo MQ

Hanzo MQ provides a managed message queue and pub/sub service built on NATS + JetStream with Hanzo extensions for durable messaging, stream processing, and event-driven architectures.

**Endpoint:** `mq.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/mq/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  -X POST https://api.hanzo.ai/v1/mq/publish \
  -d '{"subject": "orders.created", "data": {"orderId": "123"}}'
```

## Key Features

- Durable message queues
- Pub/sub with subject-based routing
- JetStream persistence
- Consumer groups and load balancing
- Dead letter queues
