# Hanzo Edge

Hanzo Edge provides serverless edge functions with global deployment, instant scaling, and zero cold starts.

**Endpoint:** `edge.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/edge/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/edge/functions
```

## Key Features

- Global edge deployment
- Zero cold starts
- TypeScript and JavaScript runtime
- Built-in KV storage access
- Custom domain routing
