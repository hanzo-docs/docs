The [iam-cpp-qt-example](https://github.com/iam/iam-cpp-qt-example) shows Hanzo IAM sign-in in a Qt desktop app.

## Run the example

### Prerequisites

- [Qt 6](https://www.qt.io/download)
- [OpenSSL](https://www.openssl.org/source/)

### Initialization

Set these 7 string parameters:

| Name           | Description                                                                                             | File                  |
| -------------- | ------------------------------------------------------------------------------------------------------- | --------------------- |
| endpoint       | Your Hanzo IAM server host/domain                                                                         | `mainwindow.h` |
| client_id      | The Client ID of your Hanzo IAM application                                                               | `mainwindow.h` |
| client_secret  | The Client Secret of your Hanzo IAM application                                                           | `mainwindow.h` |
| certificate    | The public key for the Hanzo IAM application's cert                                                       | `mainwindow.h` |
| org_name       | The name of your Hanzo IAM organization                                                                    | `mainwindow.h` |
| app_name       | The name of your Hanzo IAM application                                                                    | `mainwindow.h` |
| redirect_url   | The path of the callback URL for your Hanzo IAM application, will be `http://localhost:8080/callback` if not provided | `mainwindow.h` |

Default endpoint: `http://localhost:8000` if not set.

### Running

**Qt Creator**

1. Open `iam-cpp-qt-example.pro`
2. Set the `INCLUDEPATH` of OpenSSL in `iam-cpp-qt-example.pro`
3. Press `Ctrl + R` to start

### Listen for the callback (TCP)

```cpp
// Initialize the TcpServer object and listen on port 8080
m_tcpserver = new QTcpServer(this);
if (!m_tcpserver->listen(QHostAddress::LocalHost, 8080)) {
    qDebug() << m_tcpserver->errorString();
    close();
}
connect(m_tcpserver, SIGNAL(newConnection()), this, SLOT(on_tcp_connected()));
