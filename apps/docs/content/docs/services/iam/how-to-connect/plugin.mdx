Hanzo IAM provides plugins or middlewares for popular platforms. For CLI usage, see [Hanzo IAM CLI](/docs/how-to-connect/cli).

| Plugin | Language | Repository |
|--------|----------|------------|
| Spring Boot | Java | [iam-spring-boot-starter](https://github.com/iam/iam-spring-boot-starter) |
| Spring Boot example | Java | [iam-spring-boot-example](https://github.com/iam/iam-spring-boot-example) |
| WordPress | PHP | [wordpress-iam-plugin](https://github.com/iam/wordpress-iam-plugin) |
| Odoo | Python | [odoo-iam-oauth](https://github.com/iam/odoo-iam-oauth) |
| Django | Python | [django-iam-auth](https://github.com/iam/django-iam-auth) |
| Chrome extension | JavaScript | [iam-chrome-extension](https://github.com/iam/iam-chrome-extension) |

More official integrations: [Hanzo IAM repositories](https://github.com/orgs/iam/repositories?q=sdk+in%3Areadme&type=all&language=&sort=).
