title: Using Hanzo IAM in ShenYu
description: How to use Hanzo IAM with ShenYu
---

ShenYu has a Hanzo IAM plugin to enable the use of Hanzo IAM.

## Step 1: Deploy Hanzo IAM

Deploy Hanzo IAM first. See [Server installation](/docs/basic/server-installation).

After deployment, ensure:

- The Hanzo IAM server is running on **`http://localhost:8000`**.
- Open your preferred browser and visit **`http://localhost:7001`** to see the Hanzo IAM login page.
- Login functionality is working fine by inputting `admin` and `123`.

After the above setup, add a Hanzo IAM-based login page with the following steps.

## Step 2: Configure the Hanzo IAM application

1. **Create a new Hanzo IAM application or use an existing one**
2. **Add your redirect URL**
   ![Hanzo IAM Application Settings](/img/integration/java/shenyu/shenyu_config.png)
3. **On the certificate editing page, copy your `Certificate`**
   ![Hanzo IAM Certification Settings](/img/integration/java/shenyu/shenyu_cert.png)

## Step 3: Use the Hanzo IAM plugin in ShenYu

### 1. Configure the Hanzo IAM plugin in ShenYu

   ![Shenyu Plugin Configuration](/img/integration/java/shenyu/shenyu_configPlugin.png)

Note: As ShenYu only has a single line input box, `\n` must be added in every line of certificate.
   ![Hanzo IAM Certification Settings](/img/integration/java/shenyu/shenyu_cert2.png)

Paste it into the certificate field of the ShenYu Hanzo IAM config.

**You don't need to save it in the Hanzo IAM certificate editing page**, as it is only for copying.

### 2. Configure the ShenYu Hanzo IAM plugin

   ![Shenyu Hanzo IAM Config](/img/integration/java/shenyu/shenyu_iam.png)
   Configure the Hanzo IAM settings as needed.
