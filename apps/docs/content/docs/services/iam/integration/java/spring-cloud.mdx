In Spring Cloud, authentication is usually handled at the gateway. See [Spring Cloud Gateway](/docs/integration/java/spring-cloud-gateway). For a single service, see [iam-spring-boot-example](https://github.com/iam/iam-spring-boot-example).

Whether it's in the gateway layer or in a single service, both use the [iam-spring-boot-starter](https://github.com/iam/iam-spring-boot-starter).
