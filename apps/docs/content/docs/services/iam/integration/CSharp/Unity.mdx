## Step 1: Deploy Hanzo IAM

Deploy Hanzo IAM in **production mode**. See [Server installation](/docs/basic/server-installation). Ensure the server is reachable and you can sign in (e.g. `admin` / `123`). For a quick start, use the [Hanzo IAM demo](https://iam.hanzo.ai/).

## Step 2: Import Hanzo IAM.Client

Import [Hanzo IAM.Client](https://github.com/iam/iam-dotnet-sdk/tree/master/src/Hanzo IAM.Client) for `.NET` in the [Hanzo IAM-dotnet-sdk](https://github.com/iam/iam-dotnet-sdk).

One optional method is as follows:

- `git@github.com:iam/iam-dotnet-sdk.git`
- Run ConsoleApp in the Sample folder.
- Get the `/iam-dotnet-sdk/src/Hanzo IAM.Client/bin/Debug/net462` folder.

Import the `net462` folder (or another target framework folder) into your Unity project as shown below.

![import-sdk](/img/integration/CSharp/Unity/import-sdk.png)
