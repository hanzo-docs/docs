---
title: Internationalization
description: Hanzo IAM’s supported languages and how to contribute translations.
---

Hanzo IAM’s interface and backend messages are translated into multiple languages. Supported languages include: English, Spanish, French, German, Japanese, Chinese, Vietnamese, Portuguese, Turkish, Polish, and Ukrainian. The same set of languages is available across the product (e.g. user profiles, organization settings) for a consistent experience.

Translations are managed on [Crowdin](https://crowdin.com/project/iam-web). The project syncs with Crowdin so that community contributions are pulled in and kept up to date.

If you want to improve or add translations, contribute on [Crowdin](https://crowdin.com/project/iam-web). Your contributions help users in your language use Hanzo IAM more effectively.
