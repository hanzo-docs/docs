---
title: Local file system storage
description: Store uploaded files on the Hanzo IAM server’s filesystem.
---

The **Local File System** provider stores uploads in the Hanzo IAM `files` directory. For example, if Hanzo IAM is at `/home/user/iam`, files are stored under `/home/user/iam/files`.

## Configure the provider

![Local File configuration](/img/providers/storage/localFileConfig.png)

**Path prefix** — Optional prefix for object paths. With a prefix, files are stored under that subpath.
