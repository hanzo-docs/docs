---
title: Steam OAuth
description: Add Steam as an OAuth provider (API key only).
---

1. Sign in at [Steam Web API Key](https://steamcommunity.com/dev/revokekey) and register an API key for your Hanzo IAM domain or IP.
2. In Hanzo IAM add an **OAuth** provider, set **Type** to **Steam**. Put the API key in **Client secret**. Leave **Client ID** empty.
3. Your Steam account must have at least one game to obtain an API key.

See [Steam Web API](https://steamcommunity.com/dev).
