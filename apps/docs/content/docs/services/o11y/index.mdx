# Hanzo O11y

Hanzo O11y provides unified observability with logs, metrics, and traces in a single platform.

**Endpoint:** `o11y.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/o11y/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/o11y/logs?query=level:error&limit=100
```

## Key Features

- Unified logs, metrics, and traces
- Real-time log streaming
- Custom dashboards
- Alerting and notifications
- OpenTelemetry-compatible ingestion
