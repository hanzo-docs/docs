# Hanzo PaaS

Hanzo PaaS provides production-grade K8s-native GitOps deployments, per-org DOKS cluster management, and fleet operations.

**Endpoint:** `paas.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/paas/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/paas/deployments
```

## Key Features

- K8s-native GitOps deployments
- Per-org cluster management
- Fleet operations across clusters
- Auto-scaling and zero-downtime deploys
- Built-in CI/CD pipelines
