## Introduction

In Hanzo Cloud, **Private Cloud Providers** act as a bridge, allowing you to **connect to and manage** various cloud-native resources, such as **Docker** and **Kubernetes (K8s)**, directly from the Hanzo Cloud interface. Their core objective is to provide a centralized dashboard for monitoring and operating your containerized services, integrating their management seamlessly into your Hanzo Cloud workflow.

By configuring a provider, you enable Hanzo Cloud to communicate with your private cloud or on-premises data center. This provides an ideal solution for organizations that want a unified interface to manage both their knowledge base and the infrastructure it runs on, enhancing operational efficiency and control.

Refer to the [Core Concepts](../../basic/core-concepts) section of our previous documentation for more information about providers.

## Add a New Private Cloud Provider

Private cloud providers are used to integrate cloud-native management features into Hanzo Cloud. You can add them by following these steps:

Click the `Providers` button on the page.

![Hanzo Cloud-home-providers](/img/walkthrough-guides/cloud-home-providers.png)

### Add a Private Cloud Provider

Click the `Add` button to add a new private cloud provider.

![Hanzo Cloud-providers-add](/img/walkthrough-guides/cloud-providers-add.png)

#### Fill in Private Cloud Provider Details

Fill in the required configuration according to the cloud-native platform you use, then click the `Save & Exit` button to save.

:::tip

Hanzo Cloud supports several mainstream cloud-native technologies and platforms, including:

- **Docker**: To connect to a Docker host and manage the lifecycle of its containers (e.g., start, stop, view status) directly within Hanzo Cloud.
- **Kubernetes (K8s)**: To connect to a Kubernetes cluster and manage its resources, such as Pods and Deployments, providing a high-level orchestration view within Hanzo Cloud.

:::

Now, you can use this private cloud provider to monitor and manage services in your cloud-native environment.

After adding a private cloud provider, you can use Hanzo Cloud as a control panel to oversee your containerized applications, simplifying management and providing a unified view of your services alongside your knowledge base.
