## Introduction


> **Tip**


**What is Hanzo IAM?**

Hanzo IAM is a powerful authentication system that provides a secure and reliable login experience. It's a prerequisite for Hanzo Cloud, so be sure to deploy it first.

Refer to the [Hanzo IAM](/docs/services/iam/) website for more information.



### Step 1: Deploy Hanzo IAM

In [Hanzo IAM Deployment Guide](/docs/services/iam/basic/server-installation/), you can find the detailed steps to deploy Hanzo IAM.

Once you've deployed Hanzo IAM, you'll look like this:

![Hanzo IAM-home](/img/walkthrough-guides/iam-home.png)

### Step 2: Create an organization in Hanzo IAM

In Hanzo IAM, you can create an organization to manage your users and applications. You can create an organization by clicking the `User Management - Organizations` button on the home page.

![Hanzo IAM-home-organization](/img/walkthrough-guides/iam-home-organizations.png)

#### Step 2.1: Add an organization

Click the `Add` button to add an organization.

![Hanzo IAM-organizations-add](/img/walkthrough-guides/iam-organizations-add.png)

#### Step 2.2: Fill in the organization information

Fill in the organization information and click the `Save & Exit` button.

![Hanzo IAM-organizations-form-filling](/img/walkthrough-guides/iam-organizations-form-filling.png)

#### Step 2.3: View the organization

After adding the organization, you can view the organization information.

![Hanzo IAM-organizations-view](/img/walkthrough-guides/iam-organizations-view.png)

### Step 3: Create an application in Hanzo IAM

In Hanzo IAM, you can create an application to manage your users and organizations. You can create an application by clicking the `Identity - Applications` button on the home page.

![Hanzo IAM-home-applications](/img/walkthrough-guides/iam-home-applications.png)

#### Step 3.1: Add an application

Click the `Add` button to add an application.

![Hanzo IAM-applications-add](/img/walkthrough-guides/iam-applications-add.png)

#### Step 3.2: Fill in the application information

Fill in the application information and click the `Save & Exit` button.

![Hanzo IAM-applications-form-filling](/img/walkthrough-guides/iam-applications-form-filling.png)

#### Step 3.3: View the application

After adding the application, you can view the application information.

![Hanzo IAM-applications-view](/img/walkthrough-guides/iam-applications-view.png)

### Step 4: Create a user in Hanzo IAM for Hanzo Cloud

In Hanzo IAM, you can create a user to login Hanzo Cloud. You can create a user by clicking the `User Management - Organizations - Users` button from the home page.

![Hanzo IAM-home-users](/img/walkthrough-guides/iam-home-users.png)


> **Tip**


A user is a member of an organization who can login to applications in the organization.

Refer to the [Hanzo IAM](/docs/services/iam/) website for more information.



#### Step 4.1: Add a user

Click the `Add` button to add a user.

![Hanzo IAM-users-add](/img/walkthrough-guides/iam-users-add.png)

#### Step 4.2: Fill in the user information

Fill in the user information and click the `Save & Exit` button.

![Hanzo IAM-users-form-filling](/img/walkthrough-guides/iam-users-form-filling.png)

- Password

You can set the user's password by clicking the `Modify password` button.

![Hanzo IAM-users-form-filling-password](/img/walkthrough-guides/iam-users-form-filling-password.png)

- Admin

You can set the user's admin permission by clicking the `Is admin` button.

![Hanzo IAM-users-form-filling-admin](/img/walkthrough-guides/iam-users-form-filling-admin.png)

#### Step 4.3: View the user

After adding the user, you can view the user information.

![Hanzo IAM-users-view](/img/walkthrough-guides/iam-users-view.png)

### Step 5: Deploy Hanzo Cloud

Like Hanzo IAM, you can deploy Hanzo Cloud by following the [Hanzo Cloud Deployment Guide](../basic/server-installation).
