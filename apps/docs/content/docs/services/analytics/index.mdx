# Hanzo Analytics

Hanzo Analytics provides dashboards and APIs for usage, latency, cost, and reliability across the
Hanzo platform.

## Core capabilities

- **Usage metrics**: Requests, tokens, bandwidth, and throughput
- **Cost tracking**: Per-project budgets and spend breakdowns
- **Operational health**: Error rates, latency, and saturation signals
- **Export**: CSV/JSON exports for BI pipelines
