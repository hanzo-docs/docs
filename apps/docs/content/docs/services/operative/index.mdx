# Hanzo Operative

Hanzo Operative is a computer-use agent that allows Claude to interact with a desktop environment via screenshot, mouse, keyboard, bash, and file-editing tools.

**Endpoint:** `operative.hanzo.ai`
**Gateway:** `api.hanzo.ai/v1/operative/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  -X POST https://api.hanzo.ai/v1/operative/sessions \
  -d '{"task": "Open the browser and navigate to hanzo.ai"}'
```

## Key Features

- Full desktop environment control
- Screenshot capture and analysis
- Mouse and keyboard automation
- Bash command execution
- File editing tools
