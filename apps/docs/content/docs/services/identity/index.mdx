# Hanzo Identity

Hanzo Identity provides user profiles, organization directories, and identity resolution services
across the Hanzo platform. It complements IAM by focusing on identity data and relationships rather
than access control flows.

## What it provides

- **Profiles**: Unified user and service identity records
- **Directories**: Organization, team, and group catalogs
- **Resolution**: Link identities across services and environments
- **Auditability**: Identity history and provenance

## Relationship to IAM

IAM handles authentication and authorization. Identity powers the canonical profile and directory
data that IAM, Gateway, and other services reference.
