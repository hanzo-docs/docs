# Hanzo Bot

Hanzo Bot Hub is the skill registry and marketplace for Hanzo Bot. It provides APIs for browsing, publishing, and managing bot skills, personas, and integrations.

**Endpoint:** `hanzo.bot`
**Gateway:** `api.hanzo.ai/v1/bot/*`

## Quick Start

```bash
curl -H "Authorization: Bearer hk-your-api-key" \
  https://api.hanzo.ai/v1/bot/skills
```

## Key Features

- 743+ skills available
- Skill publishing and versioning
- Persona management
- Integration marketplace with 600+ connectors
