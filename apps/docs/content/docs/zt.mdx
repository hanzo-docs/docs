# Hanzo Zero Trust

Identity-based zero-trust networking for AI infrastructure. mTLS overlay networks, post-quantum cryptography, no VPN required. Secure connectivity across clouds, edge, and workstations.

## Highlights

- **Zero-Trust Overlay** -- Dark network with no listening ports, all connections identity-verified
- **mTLS Everywhere** -- Mutual TLS on every connection, no plaintext ever
- **Post-Quantum Cryptography** -- Dilithium lattice-based signatures for forward-secure certificates
- **No VPN Required** -- Application-embedded connectivity via lightweight SDKs
- **MCP Gateway** -- Zero-trust access to Model Context Protocol tools over the overlay
- **Private Sharing** -- Instant ephemeral tunnels via zrok for development and demos

**Console:** `zt.hanzo.ai`
**Controller API:** `zt-api.hanzo.ai`
