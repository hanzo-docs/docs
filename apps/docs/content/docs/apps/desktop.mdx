# Desktop apps

A native menubar assistant that's one keystroke away anywhere on your machine, and a connector that wires Hanzo's tools into Claude Desktop.

## Hanzo Desktop

A menubar / system-tray app with a global hotkey — **Cmd/Ctrl + Shift + Space** — for chat, quick actions, and agents over the same models and tools you use everywhere else.

| OS | Asset |
|----|-------|
| **macOS** | [`hanzo-ai-desktop-macos-v1.9.31.dmg`](https://github.com/hanzoai/extension/releases/download/v1.9.31/hanzo-ai-desktop-macos-v1.9.31.dmg) |
| **Windows** | [`hanzo-ai-desktop-windows-v1.9.31.msi`](https://github.com/hanzoai/extension/releases/download/v1.9.31/hanzo-ai-desktop-windows-v1.9.31.msi) · [`.exe`](https://github.com/hanzoai/extension/releases/download/v1.9.31/hanzo-ai-desktop-windows-v1.9.31.exe) |
| **Linux** | [`hanzo-ai-desktop-linux-v1.9.31.AppImage`](https://github.com/hanzoai/extension/releases/download/v1.9.31/hanzo-ai-desktop-linux-v1.9.31.AppImage) · [`.deb`](https://github.com/hanzoai/extension/releases/download/v1.9.31/hanzo-ai-desktop-linux-v1.9.31.deb) |

**Install**

- **macOS** — open the `.dmg` and drag **Hanzo** to Applications.
- **Windows** — run the `.msi` (or `.exe`) installer.
- **Linux** — `chmod +x hanzo-ai-desktop-linux-v1.9.31.AppImage && ./hanzo-ai-desktop-linux-v1.9.31.AppImage`, or `sudo dpkg -i hanzo-ai-desktop-linux-v1.9.31.deb`.

Launch it, press **Cmd/Ctrl + Shift + Space**, and **Sign in with Hanzo**.

## Claude Desktop

Add Hanzo's MCP tools to Anthropic's Claude Desktop with a one-file connector.

| Asset | Format |
|-------|--------|
| [`hanzo-ai-claude-v1.9.31.dxt`](https://github.com/hanzoai/extension/releases/download/v1.9.31/hanzo-ai-claude-v1.9.31.dxt) | Desktop extension (`.dxt`) |

**Install** — double-click the `.dxt` to open it with Claude Desktop, then confirm the install. Provide your `hk-` key when prompted and the Hanzo tools appear in Claude's tool list.

Newest builds are always on the [latest release](https://github.com/hanzoai/extension/releases/latest).

> **One identity, everywhere.** Sign in with Hanzo ([IAM](/docs/iam) OAuth) or paste a `hk-` [API key](/docs/api-keys). The same credential works in every Hanzo surface — see [Integrations & Extensions](/docs/apps).
