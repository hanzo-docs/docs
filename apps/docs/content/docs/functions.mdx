# Hanzo Functions

Serverless compute platform for deploying event-driven functions. Any language, auto-scaling, scale-to-zero, with deep integration into PubSub, Tasks, KMS, and IAM.

## Highlights

- **Any Language** -- Deploy functions written in Go, Python, TypeScript, Rust, or any OCI container
- **Auto-Scaling** -- Scale from zero to thousands of instances based on demand
- **Event-Driven** -- Trigger from PubSub subjects, HTTP requests, cron schedules, or Task activities
- **Secure by Default** -- IAM-scoped execution identity, KMS secrets injection, zero-trust networking
- **Observability** -- Automatic OTEL tracing, Prometheus metrics, and structured log collection

**Endpoint:** `api.hanzo.ai/v1/functions/*`
