# Hanzo API Reference

Complete API documentation for every Hanzo service. All endpoints use a single API key and route through `api.hanzo.ai`.

## Authentication

```bash
curl -H "Authorization: Bearer hk-your-api-key" https://api.hanzo.ai/v1/models
```

Get your API key at [console.hanzo.ai](https://console.hanzo.ai).

## SDKs

Use our official SDKs instead of raw HTTP:

| Language | Package | Install |
|----------|---------|---------|
| Python | [`hanzoai`](https://pypi.org/project/hanzoai/) | `pip install hanzoai` |
| TypeScript | [`hanzoai`](https://www.npmjs.com/package/hanzoai) | `npm install hanzoai` |
| Go | [`go-sdk`](https://pkg.go.dev/github.com/hanzoai/go-sdk) | `go get github.com/hanzoai/go-sdk` |
| Rust | [`hanzo-ai`](https://crates.io/crates/hanzo-pqc) | `cargo add hanzo-ai` |
| C/C++ | [`c-sdk`](https://github.com/hanzoai/c-sdk) | CMake FetchContent |

[Full SDK documentation](/docs/sdks)
