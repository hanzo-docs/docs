# Console onboarding

Connect your accounts once in the console and the same unified usage follows you
across the `dev` CLI, Hanzo Desktop, hanzo.app, and hanzo.chat.

## Connect your accounts

<Steps>
<Step>
### Open AI Accounts

Sign in to [console.hanzo.ai](https://console.hanzo.ai) and open the
**AI Accounts** section (`/ai-accounts`).
</Step>
<Step>
### Add an account

On the **Accounts** tab, choose a provider — OpenAI (Codex), Anthropic (Claude),
Hanzo, or any of the 50+ supported providers — and complete the guided
account-connect flow. It reuses your existing login (OAuth, browser session, or
API key); no passwords are stored.
</Step>
<Step>
### Repeat for every account

Link as many accounts as you like, including GPU, cloud, and integration
providers (payments, ecommerce, marketing, analytics). Each connected account
appears in the unified dashboard.
</Step>
</Steps>

## What the unified dashboard shows

For every connected account, in one view:

- **Rate limits** and time-to-reset for each window
- **Credits** and remaining balance
- **Spend** to date
- **Source** each figure was read from (OAuth, web session, CLI, or API token)

The dashboard reads through the same [`@hanzo/usage`](/docs/usage/quickstart)
core used by the CLI and desktop apps, so the numbers match everywhere.
