# Hanzo Chat

Hanzo Chat is a production-ready AI chat platform built on [LibreChat](https://librechat.ai). It ships **first-party Zen models** alongside **100+ third-party models** from OpenAI, Anthropic, Google, Meta, Mistral, Together, and Groq. Built-in MCP tool support, configurable agents, file uploads with RAG, vision, conversation history, presets, and plugins make it a complete chat solution.

Available as a hosted service at [chat.hanzo.ai](https://chat.hanzo.ai) or self-hosted via Docker.

## Quick Start

### Hosted (Recommended)

1. Go to [chat.hanzo.ai](https://chat.hanzo.ai)
2. Sign in with your Hanzo account
3. Your **$5 free credit** is applied automatically
4. Select a model and start chatting

### Self-Hosted

```bash
git clone https://github.com/hanzoai/chat
cd chat
cp .env.example .env
# Edit .env with your HANZO_API_KEY
docker compose up -d
# Open http://localhost:3081
```

### API

```bash
curl https://api.hanzo.ai/v1/chat/completions \
  -H "Authorization: Bearer $HANZO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "zen4",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## SDK Usage

### Python

```python
from openai import OpenAI

client = OpenAI(
    api_key="hk-your-api-key",
    base_url="https://api.hanzo.ai/v1"
)

response = client.chat.completions.create(
    model="zen4",
    messages=[{"role": "user", "content": "Hello, Zen."}]
)
print(response.choices[0].message.content)
```

### TypeScript

```typescript
import OpenAI from 'openai'

const client = new OpenAI({
  apiKey: process.env.HANZO_API_KEY,
  baseURL: 'https://api.hanzo.ai/v1',
})

const completion = await client.chat.completions.create({
  model: 'zen4',
  messages: [{ role: 'user', content: 'Hello, Zen.' }],
})
console.log(completion.choices[0].message.content)
```
