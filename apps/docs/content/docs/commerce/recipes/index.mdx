## Prerequisites

All recipes assume you have:

- A Hanzo account with an API key ([hanzo.ai](https://hanzo.ai))
- Node.js 18+ and npm/pnpm installed
- Basic familiarity with the [Quick Start](/docs/commerce/quickstart) guide
- The `@hanzo/commerce` SDK installed:

```bash
npm install @hanzo/commerce
```

## Recipe Structure

Each recipe follows a consistent format:

1. **Goal** -- what you will build
2. **Prerequisites** -- specific requirements beyond the basics
3. **Step-by-step implementation** -- code you can copy and run
4. **Testing** -- how to verify everything works
5. **Next steps** -- where to go from here

## Suggesting a Recipe

If you have a use case that is not covered, open an issue on the [hanzoai/commerce](https://github.com/hanzoai/commerce) repository with the `recipe-request` label.
