# hanzoai/orm

Generics-based ORM for Go. Replaces per-model boilerplate with a single `Model[T]` generic mixin -- one struct definition, zero code generation.

- **Repository:** [github.com/hanzoai/orm](https://github.com/hanzoai/orm)
- **Module:** `github.com/hanzoai/orm`
- **pkg.go.dev:** [pkg.go.dev/github.com/hanzoai/orm](https://pkg.go.dev/github.com/hanzoai/orm)
- **Version:** v0.2.0
- **License:** MIT

## Install

```bash
go get github.com/hanzoai/orm@latest
```

## Packages

| Package | Description |
|---------|-------------|
| `orm` | Core `Model[T]`, registration, hooks, cache, serialization |
| `orm/db` | Database interfaces + SQLite driver + ZAP binary protocol driver |
| `orm/val` | Struct field validation |
| `orm/internal/json` | JSON encode/decode helpers |
| `orm/internal/reflect` | Reflection utilities |
