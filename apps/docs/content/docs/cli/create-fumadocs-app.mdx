## Overview

Despite the classical CLI usage, you can also use it in scripts like:

```npm
npm install create-hanzo-docs-app
```

```ts
import { create } from 'create-hanzo-docs-app';

await create({
  outputDir: 'my-app',
  template: '+next+fuma-docs-mdx',
  packageManager: 'pnpm',
});
```
