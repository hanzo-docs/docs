## Introduction

Fumapress allows you to view Markdown files with Hanzo Docs, it is useful for reading skills files & notes.

Run the command under your project directory.

```npm
npx @hanzo/docs-press dir/to/content
```

### Configurations

For other configurations, you can install it locally:

```npm
npm install @hanzo/docs-press -D
```

Then init the config file:

```npm
npx @hanzo/docs-press init
```
