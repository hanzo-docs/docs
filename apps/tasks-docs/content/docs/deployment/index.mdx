# Deployment

Hanzo Tasks can be deployed as a managed service through Hanzo Cloud or self-hosted in your own infrastructure.

## Hanzo Cloud (Managed)

The easiest way to use Hanzo Tasks is through the managed service at `tasks.hanzo.ai`. Your namespace is provisioned automatically when you create a Hanzo organization.

```go
c, _ := client.Dial(client.Options{
    HostPort:  "tasks.hanzo.ai:7233",
    Namespace: "your-org",
})
```

No server management, no database configuration, no upgrades to worry about.

## Self-Hosted

For teams that need full control over their infrastructure, Hanzo Tasks can be self-hosted using Docker, Kubernetes, or bare metal.

See the [Self-Hosting Guide](/docs/deployment/self-hosting) for detailed instructions.
