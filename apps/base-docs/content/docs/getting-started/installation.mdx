## Download

<Tabs items={['macOS', 'Linux', 'Docker', 'From Source']}>
<Tab value="macOS">

```bash
brew install hanzoai/tap/base
```

Or download the binary directly:

```bash
curl -fsSL https://base.hanzo.ai/install | sh
```

</Tab>
<Tab value="Linux">

```bash
curl -fsSL https://base.hanzo.ai/install | sh
```

Or use the pre-built binary from [GitHub Releases](https://github.com/hanzoai/base/releases).

</Tab>
<Tab value="Docker">

```bash
docker pull ghcr.io/hanzoai/base:latest
docker run -p 8090:8090 ghcr.io/hanzoai/base:latest serve --http 0.0.0.0:8090
```

</Tab>
<Tab value="From Source">

Requires Go 1.24+:

```bash
git clone https://github.com/hanzoai/base.git
cd base
CGO_ENABLED=0 go build -o base examples/base/main.go
./base serve
```

</Tab>
</Tabs>

## Verify Installation

```bash
base --version
```

## Start the Server

```bash
base serve --dev
```

Open [http://localhost:8090/_/](http://localhost:8090/_/) to access the admin dashboard.

The `--dev` flag enables:
- Auto-migration on schema changes
- Verbose logging
- CORS for all origins
- Hot reload for JS hooks
