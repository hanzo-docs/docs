## Authentication

Hanzo Dev supports multiple authentication methods.

### API Key

Set your API key via environment variable:

```bash
export OPENAI_API_KEY="sk-..."
```

Or login interactively:

```bash
dev login
```

### Login with API Key from stdin

```bash
printenv OPENAI_API_KEY | dev login --with-api-key
```

### Check Status

```bash
dev login status
```

### Logout

```bash
dev logout
```

### Environment Variables

| Variable | Description |
| --- | --- |
| `OPENAI_API_KEY` | OpenAI API key |
| `CODE_HOME` | Override config directory (default: `~/.hanzo`) |
| `CODEX_HOME` | Legacy config directory (fallback) |
| `CODEX_SECURE_MODE` | Enable security hardening when set to `"1"` |
