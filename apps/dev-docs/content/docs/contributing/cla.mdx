## Contributor License Agreement (CLA)

All contributors **must** accept the CLA. The process is lightweight:

1. Open your pull request
2. Paste the following comment (or reply `recheck` if you've signed before):

```text
I have read the CLA Document and I hereby sign the CLA
```

3. The CLA-Assistant bot records your signature and marks the status check as passed

No special Git commands, email attachments, or commit footers required.

## Security & Responsible AI

Discovered a vulnerability or have concerns about model output? Please email **security@hanzo.ai** and we will respond promptly.
