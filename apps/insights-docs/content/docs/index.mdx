# Hanzo Insights

Hanzo Insights is a self-hosted product analytics platform with 45+ built-in products for understanding your users, shipping better features, and growing your product.

## Quick Links

- [Getting Started](/docs/getting-started) — Install and configure Insights
- [Events API](/docs/api/events) — Send events from any platform
- [Feature Flags](/docs/products/feature-flags) — Ship with confidence
- [Session Replay](/docs/products/replay) — Watch real user sessions
- [A/B Testing](/docs/products/experiments) — Run experiments

## Products

Insights ships with 45 built-in products organized into categories:

- **Analytics** — Product analytics, web analytics, funnels, cohorts
- **Flags & Experiments** — Feature flags, A/B tests, early access
- **Session Recording** — Replay, desktop recordings, AI summaries
- **User Engagement** — Surveys, product tours, conversations
- **Data Platform** — Data warehouse, CDP, batch exports
- **AI Analytics** — LLM tracking, error monitoring, signals
