# Products

Hanzo Insights ships with 45 built-in products covering every aspect of product analytics.

## Analytics

- **Product Analytics** — Funnels, retention, paths, and cohort analysis
- **Web Analytics** — Lightweight page tracking with privacy controls
- **Customer Analytics** — Customer journey and LTV tracking
- **Dashboards** — Custom dashboards and data visualization
- **Live Debugger** — Real-time event stream for debugging

## Session Recording

- **Session Replay** — Full session recordings with privacy masking
- **Desktop Recordings** — Desktop and mobile screen capture
- **Session Summaries** — AI-powered session analysis

## Feature Management

- **Feature Flags** — Instant rollouts, rollbacks, and targeting
- **Early Access Features** — Beta program management
- **Experiments (A/B Testing)** — Statistically rigorous experimentation

## User Engagement

- **Surveys** — In-app NPS, CSAT, and custom surveys
- **Product Tours** — Guided onboarding flows
- **Conversations** — User feedback and interviews
- **User Interviews** — Structured research sessions

## Data Platform

- **Data Warehouse** — Direct SQL access to all your data
- **Data Modeling** — Custom event schemas and transforms
- **Batch Exports** — Export to BigQuery, Snowflake, S3
- **CDP** — Customer Data Platform with identity resolution

## AI & Observability

- **LLM Analytics** — Track prompts, completions, latency, and cost
- **AI Insights** — Automated anomaly detection
- **Error Tracking** — Frontend and backend error monitoring
- **Signals** — Configurable alerts and notifications
