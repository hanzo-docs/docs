Hanzo Platform is a unified cloud deployment platform that lets you deploy and manage applications across Kubernetes clusters, Docker Swarm services, Docker Compose stacks, and virtual machines — all from a single dashboard.

## Key Features

- **Multi-Orchestrator Support** — Kubernetes, Docker Swarm, and Docker Compose
- **VM Management** — Launch and manage VMs on AWS, DigitalOcean, and Hetzner
- **Git-Powered Builds** — Push to deploy with Tekton pipelines and Kaniko
- **Multi-Cluster Fleet** — Manage all your clusters from one place
- **Enterprise RBAC** — Organizations, teams, roles, and environment protection
- **Real-Time Monitoring** — Stream logs, view metrics, and track deployments
