All models are available via the OpenAI-compatible API at `api.hanzo.ai` and `api.zen.hanzo.ai`. Pricing is set at 3x upstream inference cost for transparent, sustainable pricing.

## API Usage

All models are accessed via the OpenAI-compatible API. API key prefix: `hk-`.

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.hanzo.ai/v1",
    api_key="hk-your-api-key",
)

response = client.chat.completions.create(
    model="zen4",
    messages=[{"role": "user", "content": "Hello!"}],
)
print(response.choices[0].message.content)
```

## Open Weights

All locally-runnable models are available as open weights on [HuggingFace](https://huggingface.co/zenlm) in multiple formats:

| Format | Use Case | Platform |
|--------|----------|----------|
| **SafeTensors** | Full precision, transformers | All |
| **GGUF** | Quantized, llama.cpp/Ollama | All |
| **MLX** | Apple Silicon optimized | macOS |
