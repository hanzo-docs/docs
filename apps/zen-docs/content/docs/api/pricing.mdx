All pricing is **per 1 million tokens**. Zen models are priced at **3x upstream inference cost** for transparent, sustainable pricing. Pay only for what you use.

## Pricing Methodology

Zen pricing is set at **3x the upstream Fireworks inference cost** for each model. This covers Hanzo infrastructure, identity optimization, API management, and margin. The upstream model and provider are disclosed on every model page for full transparency.

## Billing

- **Prepaid**: Add credits via [console.hanzo.ai](https://console.hanzo.ai)
- **Minimum balance**: $1.00 required to make API calls
- **Usage tracking**: Real-time usage visible in console
- **No surprise bills**: Service stops when credits are depleted
