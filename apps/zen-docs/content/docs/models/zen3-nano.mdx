# zen3-nano

**Edge**

Ultra-lightweight model for edge deployment and low-latency tasks. Built on Qwen3-4B by Alibaba Qwen -- the smallest model in the Zen lineup.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `zen3-nano` |
| **Upstream** | Qwen3-4B by Alibaba Qwen |
| **Parameters** | 4B |
| **Architecture** | Dense Transformer |
| **Context Window** | 40K tokens |
| **Tier** | pro |
| **Input Price** | $0.30 / 1M tokens |
| **Output Price** | $0.30 / 1M tokens |

## Capabilities

- Ultra-low latency inference
- Edge and IoT deployment
- Mobile applications
- High-volume classification tasks
- Basic chat and instruction following
- 40K context window

## API Usage

```bash
curl https://api.hanzo.ai/v1/chat/completions \
  -H "Authorization: Bearer $HANZO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "zen3-nano",
    "messages": [{"role": "user", "content": "Classify this text as positive or negative: Great product!"}]
  }'
```

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.hanzo.ai/v1",
    api_key="hk-your-api-key",
)

response = client.chat.completions.create(
    model="zen3-nano",
    messages=[{"role": "user", "content": "Classify this text as positive or negative: Great product!"}],
)
print(response.choices[0].message.content)
```

## Upstream

zen3-nano is built on open-source weights from **Qwen3-4B** by Alibaba Qwen. At $0.30/MTok for both input and output, it is the cheapest model in the Zen lineup -- ideal for high-volume tasks where latency matters more than peak capability. Pricing is set at 3x upstream inference cost.
