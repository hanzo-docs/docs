# zen-scribe

Speech-to-text model for audio transcription, meeting notes, and voice processing.

## Overview

zen-scribe handles audio transcription with support for multiple languages, speaker diarization, and timestamp generation. Part of the Zen multimodal family alongside zen3-omni.

## Capabilities

- **Multi-language transcription**: Support for 50+ languages
- **Speaker diarization**: Identify and label different speakers
- **Timestamps**: Word-level and segment-level timing
- **Long-form audio**: Process hours of audio content
- **Real-time streaming**: Low-latency transcription mode

## Quick Start

```python
from transformers import pipeline

transcriber = pipeline("automatic-speech-recognition", model="zenlm/zen-scribe")
result = transcriber("audio.wav", return_timestamps=True)
print(result["text"])
```

### Via Hanzo API

```bash
curl https://api.hanzo.ai/v1/audio/transcriptions \
  -H "Authorization: Bearer $HANZO_API_KEY" \
  -F file="@audio.wav" \
  -F model="zen-scribe"
```
