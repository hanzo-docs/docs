**Maximum Reasoning**

The highest-capability Zen reasoning model. Built on GLM-5 (thinking mode) by Zhipu AI with 744B MoE architecture and extended chain-of-thought for complex problems.

## API Usage

```bash
curl https://api.hanzo.ai/v1/chat/completions \
  -H "Authorization: Bearer $HANZO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "zen4-ultra",
    "messages": [{"role": "user", "content": "Prove that the square root of 2 is irrational."}]
  }'
```

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.hanzo.ai/v1",
    api_key="hk-your-api-key",
)

response = client.chat.completions.create(
    model="zen4-ultra",
    messages=[{"role": "user", "content": "Prove that the square root of 2 is irrational."}],
)
print(response.choices[0].message.content)
```

## Upstream

zen4-ultra is built on open-source weights from **GLM-5 (thinking)** by Zhipu AI. The thinking variant enables explicit chain-of-thought reasoning for deeper problem solving. Pricing is set at 3x upstream inference cost.
