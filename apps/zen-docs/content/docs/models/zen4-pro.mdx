# zen4-pro

**High Capability**

Efficient MoE model for demanding workloads. Built on Qwen3-Next-80B-A3B by Alibaba Qwen -- 80B total parameters with only 3B active per token.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `zen4-pro` |
| **Upstream** | Qwen3-Next-80B-A3B by Alibaba Qwen |
| **Parameters** | 80B (3B active) |
| **Architecture** | MoE |
| **Context Window** | 131K tokens |
| **Tier** | ultra |
| **Input Price** | $2.70 / 1M tokens |
| **Output Price** | $2.70 / 1M tokens |

## Capabilities

- High-quality reasoning with MoE efficiency
- Strong performance on demanding benchmarks
- Balanced cost-to-quality for production workloads
- 131K context window for extended conversations
- Hybrid Gated DeltaNet + Gated Attention architecture

## API Usage

```bash
curl https://api.hanzo.ai/v1/chat/completions \
  -H "Authorization: Bearer $HANZO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "zen4-pro",
    "messages": [{"role": "user", "content": "Analyze the trade-offs between microservices and monoliths."}]
  }'
```

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.hanzo.ai/v1",
    api_key="hk-your-api-key",
)

response = client.chat.completions.create(
    model="zen4-pro",
    messages=[{"role": "user", "content": "Analyze the trade-offs between microservices and monoliths."}],
)
print(response.choices[0].message.content)
```

## Upstream

zen4-pro is built on open-source weights from **Qwen3-Next-80B-A3B** by Alibaba Qwen. The MoE architecture keeps only 3B parameters active per token, delivering strong performance at efficient inference cost. Pricing is set at 3x upstream inference cost.
