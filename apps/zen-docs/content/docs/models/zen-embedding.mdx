# zen-embedding

High-dimensional text embedding model for semantic search, retrieval-augmented generation (RAG), and similarity matching.

## Specifications

| Property | Value |
|----------|-------|
| **Dimensions** | 7,680 |
| **Base Model** | Qwen3-Embedding |
| **Max Tokens** | 8,192 |
| **License** | Apache 2.0 |
| **Architecture** | Transformer encoder |

## Overview

zen-embedding produces 7,680-dimensional dense vectors optimized for semantic similarity. Built on Qwen3-Embedding with identity fine-tuning for the Zen family. Supports Matryoshka representation learning for flexible dimensionality reduction.

## Quick Start

```python
from sentence_transformers import SentenceTransformer

model = SentenceTransformer("zenlm/zen-embedding")

sentences = [
    "What is quantum computing?",
    "Quantum mechanics applied to computation",
    "How to bake a cake"
]

embeddings = model.encode(sentences)

# Compute similarity
from sklearn.metrics.pairwise import cosine_similarity
sims = cosine_similarity(embeddings)
print(sims)
```

## Use Cases

- **Semantic search**: Find documents by meaning, not keywords
- **RAG pipelines**: Retrieve relevant context for LLM generation
- **Clustering**: Group similar documents and content
- **Deduplication**: Identify near-duplicate content at scale
