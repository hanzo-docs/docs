# zen-designer

Image generation model specialized for UI design, component mockups, and visual asset creation.

## Overview

zen-designer generates design assets from text descriptions. Optimized for UI/UX workflows — component layouts, color schemes, icon design, and marketing visuals.

## Capabilities

- **UI mockups**: Generate interface designs from descriptions
- **Component design**: Create buttons, cards, forms, and layouts
- **Brand assets**: Logos, icons, and marketing materials
- **Style transfer**: Apply design systems to generated content
- **Responsive variants**: Generate designs at multiple viewport sizes

## Quick Start

```python
from diffusers import StableDiffusionPipeline

pipe = StableDiffusionPipeline.from_pretrained("zenlm/zen-designer")

image = pipe(
    "A clean, modern dashboard UI with dark mode, showing analytics charts and user metrics",
    num_inference_steps=30
).images[0]

image.save("dashboard.png")
```

### Via Hanzo API

```bash
curl https://api.hanzo.ai/v1/images/generations \
  -H "Authorization: Bearer $HANZO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "zen-designer",
    "prompt": "Modern SaaS landing page hero section with gradient background"
  }'
```
