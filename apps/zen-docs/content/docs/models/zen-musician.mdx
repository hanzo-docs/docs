# zen-musician

**Music Generation**

A music generation model that composes original tracks with multi-instrument arrangements. Supports genre control, mood specification, and instrument selection for professional-quality audio output.

> This model is coming soon. Join the waitlist at [hanzo.chat](https://hanzo.chat).

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `zen-musician` |
| **Architecture** | Audio Transformer |
| **Output** | WAV/MP3, up to 5 minutes |
| **Sample Rate** | 44.1 kHz stereo |
| **Status** | Coming Soon |
| **HuggingFace** | -- |

## Capabilities

- Text-to-music generation
- Multi-instrument arrangement
- Genre and mood control
- Tempo and key specification
- Loop and stem generation
- Style transfer from reference audio
