# zen-musician

Music generation model for creating audio content from text descriptions.

## Overview

zen-musician generates music, sound effects, and audio from text prompts. Supports multiple genres, instruments, and moods for creative audio production.

## Capabilities

- **Text-to-music**: Generate music from natural language descriptions
- **Genre control**: Specify genre, tempo, key, and instrumentation
- **Sound effects**: Generate ambient sounds and SFX
- **Variable length**: Generate clips from 5 seconds to 5 minutes
- **Stem separation**: Output individual instrument tracks

## Quick Start

```python
from transformers import pipeline

generator = pipeline("text-to-audio", model="zenlm/zen-musician")

audio = generator(
    "Upbeat electronic music with synth pads and a driving beat, 120 BPM",
    forward_params={"max_new_tokens": 1024}
)

import soundfile as sf
sf.write("output.wav", audio["audio"][0], audio["sampling_rate"])
```

### Via Hanzo API

```bash
curl https://api.hanzo.ai/v1/audio/generations \
  -H "Authorization: Bearer $HANZO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "zen-musician",
    "prompt": "Calm ambient piano melody for a meditation app"
  }'
```
