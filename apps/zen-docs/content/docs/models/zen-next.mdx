# zen-next

zen-next refers to models built on the Qwen3-Next architecture — a hybrid Gated DeltaNet + Gated Attention + MoE design used in zen4-pro-max and zen4-coder.

## Architecture

The Qwen3-Next base introduces a hybrid attention mechanism:

| Component | Purpose |
|-----------|---------|
| **Gated DeltaNet** | Linear-time recurrent attention for long sequences |
| **Gated Attention** | Standard attention with gating for quality-critical layers |
| **MoE routing** | Mixture-of-Experts for parameter efficiency |

This hybrid approach enables 256K context with only 3B active parameters from 80B total.

## Models Using zen-next Architecture

| Model | Parameters | Active | Context |
|-------|-----------|--------|---------|
| **zen4-pro-max** | 80B MoE | 3B | 256K |
| **zen4-coder** | 80B MoE | 3B | 256K |

## Key Properties

- **256K context window** via hybrid linear/quadratic attention
- **3B active parameters** from 80B total — efficient inference
- **Sub-linear memory scaling** for long sequences
- **Apache 2.0 license** for all zen-next based models
