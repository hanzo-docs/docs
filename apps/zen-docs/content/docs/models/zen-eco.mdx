# zen-eco

Ultra-efficient models optimized for edge, mobile, and resource-constrained environments.

## Overview

The zen-eco family provides the smallest Zen models for deployment where compute and memory are limited. Built on efficient architectures with aggressive quantization support.

## Models

| Model | Parameters | Context | Use Case |
|-------|-----------|---------|----------|
| **zen-nano** | 0.6B | 32K | Embedded / IoT |
| **zen4-mini** | 4B | 32K | Mobile / Edge |
| **zen-eco** | 4B | 32K | Resource-constrained servers |

## Key Features

- **Sub-2GB memory** for quantized nano models
- **Mobile-ready**: Runs on phones via llama.cpp or MLC-LLM
- **GGUF quantization**: Q4_K_M, Q5_K_M, Q8_0 variants
- **MLX support**: Optimized for Apple Silicon (M1-M4)

## Quick Start

### Run on mobile with Ollama

```bash
ollama run zen-nano
```

### GGUF with llama.cpp

```bash
./llama-cli -m zen-nano-Q4_K_M.gguf -p "Hello" -n 128
```
