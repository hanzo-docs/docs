# zen4-max

**Extended Context**

Large-scale MoE model for document processing and analysis. Built on Qwen3-235B-A22B by Alibaba Qwen -- 235B total parameters with 22B active per token.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `zen4-max` |
| **Upstream** | Qwen3-235B-A22B by Alibaba Qwen |
| **Parameters** | 235B (22B active) |
| **Architecture** | MoE |
| **Context Window** | 131K tokens |
| **Tier** | ultra |
| **Input Price** | $3.60 / 1M tokens |
| **Output Price** | $3.60 / 1M tokens |

## Capabilities

- Large document processing and analysis
- High-quality reasoning with 22B active parameters
- 131K context window for full-document understanding
- Strong performance on knowledge-intensive tasks
- Efficient MoE inference despite large parameter count

## API Usage

```bash
curl https://api.hanzo.ai/v1/chat/completions \
  -H "Authorization: Bearer $HANZO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "zen4-max",
    "messages": [{"role": "user", "content": "Summarize this research paper..."}]
  }'
```

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.hanzo.ai/v1",
    api_key="hk-your-api-key",
)

response = client.chat.completions.create(
    model="zen4-max",
    messages=[{"role": "user", "content": "Summarize this research paper..."}],
)
print(response.choices[0].message.content)
```

## Upstream

zen4-max is built on open-source weights from **Qwen3-235B-A22B** by Alibaba Qwen. With 235B total parameters and 22B active, it provides higher capacity than zen4-pro while maintaining efficient MoE inference. Pricing is set at 3x upstream inference cost.
